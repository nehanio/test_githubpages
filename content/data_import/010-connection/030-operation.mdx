# 接続情報の編集

## 編集方法

対象の接続情報ページを開き、作成時と同様の流れで項目を編集し保存することで編集ができます。  
ただし、一部の項目は初回保存後に画面に表示されなくなり、再編集が不可能になります。  
再編集不可項目を変更したい場合は、新しく接続情報を作り直す必要があります。  
なお、再編集不可対象については、各接続情報ページで確認できます。

## 編集の影響

接続情報を編集しても、データソース、および SQL ビルダーに設定した接続情報が解除されることは有りません。
