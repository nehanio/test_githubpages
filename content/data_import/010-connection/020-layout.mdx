# データタイプ選択画面

[!image]

## 1. データタイプ選択フィールド

いずれかをクリックすることで、対象のデータタイプの接続情報入力画面に遷移します。

## 2. データタイプ検索ボックス

一覧からデータタイプを絞り込むことができます。

- 大文字小文字を区別しません
- スペースを含めた完全・部分一致検索はできません
- 検索実行操作を必要とせず、入力に応じて絞り込んだ結果を動的に表示します

## 3. その他のデータタイプ参照リンク

青字部をクリックすると、対応可能全データタイプが掲載されているページが別タプで開きます。（https://www.cdata.com/jp/drivers/）

# 接続情報入力画面

[!image]

## 1. 接続情報名入力フィールド

接続情報に任意の名前をつけることができます。  
この名前が`トップページ > 接続情報`に一覧で表示されます。

## 2. 接続情報入力フィールド

各データイプごとに指定された接続情報を入力するフィールドです。

## 3. 保存ボタン

このボタンをクリックすることで接続情報を保存できます。
