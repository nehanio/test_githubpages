# 概要

nehan に連携して取り込んだデータのことを「データソース」と呼びます。  
自社が保有するデータを分析するためには、まずデータソースとして nehan に連携する必要があります。  
データソースは「分析プロジェクト」で前処理・分析などを行うことができます。  
なお、ファイルアップロード以外のデータソース作成には接続情報が必要です。

---

# 詳細

データソースとして御社のデータを nehan に登録する際、下記の処理を行います。  
また、データソース更新時も同様です。

1. データソースの設定で指定した御社のデータを、nehan 環境内にアップロードする
2. アップロードされたデータを読み込み、nehan データソースが作成される
