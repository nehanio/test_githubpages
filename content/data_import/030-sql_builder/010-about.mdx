# 概要

SQL ビルダーはデータソースを SQL 記述不要で作成する機能です。  
前提として「データベース系」一部の「その他サービス系」について、SQL を記述しその結果をデータソースにできます。  
が、SQL を書けない、もしくは SQL を書くのが面倒な場合、一切 SQL を書かずとも SQL を作成できる補助機能が SQL ビルダーです。

## 操作

分析プロジェクトと同様のノード操作を用いて、テーブル上のデータ操作を行うことができます。

## SQL 生成

作成したフローを nehan が解釈し、自動で SQL を作成します。

## データソース作成

エクスポートノードを接続し、データソースを作成できます。
