# 概要

nehan は、Amazon Web Servise（AWS）を用いて開発しており、下記の構成でサービスを提供しております。

```mermaid
graph LR
ユーザ PC -- 参照 --> EC2(API/分析サーバ)
EC2(API/分析サーバ) -- 相互疎通 --> Aurora MySQL(アプリケーションの設定保存)
EC2(API/分析サーバ) -- 相互疎通 --> ElasticCache Redis(ジョブキュー管理)
EC2(API/分析サーバ) -- 相互疎通 --> S3(分析データの保管)
EC2(API/分析サーバ) -- アプリケーションログ保存 --> CloudWatch Logs
EC2(ビジュアライズ、ダッシュボード上のデータ操作) -- 元データ参照 --> S3(分析データの保管)
```
