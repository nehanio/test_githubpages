# 概要

Hyper nehan はビッグデータ処理に特化したオプション機能です。

- データが大きすぎてデータソース作成に時間がかかる
- データが大きすぎて分析プロジェクト内のデータ処理に時間がかかる

といった課題を解決し、実行時間を短縮できます。

## 対応機能の見分け方

Hyper nehan 対応の機能には、

- データソースのデータタイプ
- 分析プロジェクトのデータ処理機能

に、下記アイコン（Hyper アイコン）が付与されています。

[!image]

## Hyper nehan 処理が行われた状態

実際に Hyper nehan 処理が行われると、Hyper アイコンが点灯します。

### Hyper nehan 処理が行われたデータソース

データソース一覧のデータタイプ横にアイコンが点灯します。  
[!image]

### Hyper nehan 処理が行われたノード

分析プロジェクトワークスペース内のノード内にアイコンが点灯します。  
[!image]
