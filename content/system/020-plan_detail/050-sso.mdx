# 概要

ユーザーが nehan にログインする際、御社内で利用している認証サービスを用いてログインできます。  
この機能は有償にてご提供可能です。

---

# 詳細

## 提供実績があるもの

- [HENNGE ONE](https://teachme.jp/35563/manuals/10005198/)
- OneLogin
- Google Login

## 連携準備

### 御社側でご用意いただくもの

- ID プロバイダのエンティティ ID
- SAML SSO の URL
- 証明書(—–BEGIN CERTIFICATE—– で開始し、—–END CERTIFICATE—–で終了するもの)

### nehan 側からご提供し、設定いただくもの

| 設定項目        | 値                                                  |
| --------------- | --------------------------------------------------- |
| ACS URL         | https://nehan-51472.firebaseapp.com/__/auth/handler |
| エンティティ ID | nehan                                               |
| NameID          | Email アドレス                                      |
