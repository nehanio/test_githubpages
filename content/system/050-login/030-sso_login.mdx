# 概要

「シングルサインオン」オプション適用組織の場合のログイン操作を紹介します。  
なお、ログインには組織固有の「テナント ID」が必要です。  
「テナント ID」は株式会社 nehan が組織ごとに発行する文字列です。  
この ID は発行時に nehan マネージャー権限のユーザに通知しています。  
ご不明な場合は御社内でご確認ください。
