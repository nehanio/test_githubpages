# 概要

分析プロジェクトでの実行、データソース読み込みの完了時に、ブラウザの通知を受け取ることができます。  
ユーザは、この機能の ON/OFF を任意で切り替えることができます。  
※推奨ブラウザである Google Chrome を前提に記載しています。

---

# 実行通知機能

実行対象の分析プロジェクト、およびデータソースのページを開いている状態で実行完了すると、ブラウザの機能を用いて実行結果を通知します。  
これにより、nehan を開いたブラウザのタブを監視し続けずとも、実行完了を確認できます。  
※ブラウザ側の設定などは[Google Chrome ヘルプ](https://support.google.com/chrome/answer/3220216?hl=ja&co=GENIE.Platform%3DDesktop)をご確認ください。

# テスト

## 静的テスト

| id  | テストケース                                                              |
| --- | ------------------------------------------------------------------------- |
| 1   | データソース実行通知テキストが「xxx」であることを確認する                 |
| 2   | 分析プロジェクトのノード単体実行通知テキストが「xxx」であることを確認する |
| 3   | 分析プロジェクトのノード後続実行通知テキストが「xxx」であることを確認する |
| 4   | 分析プロジェクトのノード一括実行通知テキストが「xxx」であることを確認する |

## 動的テスト

| id  | テストケース                                                                                                        |
| --- | ------------------------------------------------------------------------------------------------------------------- |
| 1   | 通知 ON にした状態で、データソース xxx を「読み込んで保存」してブラウザ通知が行われることを確認する                 |
| 2   | 通知 OFF にした状態で、データソース xxx を「読み込んで保存」してブラウザ通知が行われないことを確認する              |
| 3   | 通知 ON にした状態で、分析プロジェクト xxx のワークフロー を「一括実行」してブラウザ通知が行われることを確認する    |
| 4   | 通知 OFF にした状態で、分析プロジェクト xxx のワークフロー を「一括実行」してブラウザ通知が行われないことを確認する |
